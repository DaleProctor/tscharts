
How to Use This ZIP File
------------------------

This directory contains binary releases for Windows 10.

To use, copy to a Windows 10 machine, unpack, then in a command window, cd
to the location where the file was unpacked, cd into xrayuploader, then
type xrayuploader -h for more information.
